# LioraLang

> Языки: [English (default)](README.md) | **Русский** | [Polski](README.pl.md)

Перестань собирать слова, которые забываются через день.

**LioraLang — это приложение, после которого сложно вернуться к старому способу учить язык.**
Оно превращает ваши реальные слова в стабильную ежедневную систему: собственные колоды, мощные флешкарточки, spaced repetition и полный контроль над данными офлайн.

Если вы действительно хотите говорить и понимать быстрее, LioraLang становится ежедневным инструментом, а не «очередным приложением на неделю».

## Почему без него уже не хочется учить

- Вы учите **свои слова**, а не случайный контент из шаблонных списков.
- Данные хранятся локально: без обязательного облака и привязки к серверу.
- Колоды под реальные цели: работа, интервью, переезд, путешествия, экзамены.
- Большие карточки с переворотом удерживают внимание и темп.
- Колоды можно импортировать/экспортировать и делиться ими с другими.

## Что внутри

- Режим `Learn` с крупными карточками и анимацией переворота.
- Система `Decks`: создать, редактировать, переименовать, удалить, импортировать, экспортировать.
- Страницы колод с фильтрацией и пагинацией.
- Редактор колоды: языки, описание, теги, таблица слов.
- Раздел `Progress` с современными графиками.
- Светлая и тёмная темы.
- Desktop-first архитектура с локальной SQLite-базой.

## Для кого

- Для тех, кто устал от «универсальных» курсов.
- Для тех, кому нужен управляемый рост словарного запаса.
- Для тех, кому важны приватность и владение своими данными.
- Для тех, кто строит долгую систему, а не разовую мотивацию.

## Типичный сценарий

1. Создаёте колоду под конкретную цель.
2. Добавляете слова, переводы, теги и примеры из жизни.
3. Каждый день проходите сессию в `Learn`.
4. Улучшаете, экспортируете и делитесь колодами.
5. Повторяете, пока словарь не становится автоматическим навыком.

## Формат импорта/экспорта

LioraLang поддерживает пакет колоды с метаданными и словами.
Минимально обязательное поле слова: `source`.

```json
{
  "format": "lioralang.deck",
  "version": 1,
  "deck": {
    "name": "Travel Basics",
    "description": "Core words for travel",
    "sourceLanguage": "English",
    "targetLanguage": "Polish",
    "tertiaryLanguage": "Russian",
    "tags": ["travel", "starter"]
  },
  "words": [
    {
      "id": "w1",
      "source": "apple",
      "target": "jablko",
      "tertiary": "яблоко",
      "level": "A1",
      "part_of_speech": "noun",
      "tags": ["food", "basic"],
      "examples": ["I eat an apple."]
    }
  ]
}
```

## Опционально: самостоятельная сборка

### Требования

- Node.js 20+
- pnpm 10+

### Установка

```bash
pnpm install
```

### Запуск

```bash
pnpm run dev
```

### Сборка инсталляторов

```bash
pnpm run build
pnpm run dist:test
```

Инсталляторы будут в папке `release/`.

## Troubleshooting

### Electron failed to install correctly

```bash
pnpm approve-builds
pnpm install
```

### better-sqlite3 NODE_MODULE_VERSION mismatch

```bash
pnpm run rebuild:native
```

### Белый экран в packaged-версии (`index-*.js/css` не найдены)

```bash
pnpm run build
pnpm run dist:test
```
